# Hello World API

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd hello-world-api
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the API:
   ```bash
   npm start
   ```

4. Test the API:
   ```bash
   npm test
   ```

## API Endpoint

- **GET /hello**: Returns a JSON response with a greeting message.
