# Hello World API

## Setup Instructions

1. Clone the repository.
2. Navigate to the project directory.
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
5. Test the endpoint:
   - Access `http://localhost:5000/hello` to see the JSON response.
